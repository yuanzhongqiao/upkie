<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Upkie</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">是一款完全开源的自平衡轮式双足机器人。</font><font style="vertical-align: inherit;">它有用于平衡的轮子，以及用于越野或穿越不平坦地形的腿。</font><font style="vertical-align: inherit;">Upkies 设计为可在家中使用在线订购的工具和组件进行构建，例如 mjbots 执行器。</font><font style="vertical-align: inherit;">运动控制在机器人的 Raspberry Pi 上运行。</font></font></p>
